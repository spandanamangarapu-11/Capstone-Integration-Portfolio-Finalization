# Capstone Integration & Portfolio Finalization

## Task 5 - ApexPlanet Data Analytics Internship

### Project Overview
This capstone project consolidates all internship learnings into a single portfolio project using the Superstore Sales Dataset. The analysis focuses on data cleaning, exploratory data analysis (EDA), business intelligence, and visualization to uncover key business insights and sales trends.

---

## Objectives
- Perform end-to-end data analysis on retail sales data.
- Identify sales and profit trends.
- Analyze customer, product, and regional performance.
- Generate actionable business insights through visualizations.
- Showcase data analytics skills in a professional portfolio.

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Dataset
- Sample Superstore Dataset
- Contains sales, profit, customer, product, and regional information.

---

## Key Analysis Performed
- Data Cleaning & Preprocessing
- Missing Value Analysis
- Sales & Profit Analysis
- Category and Sub-Category Analysis
- Region-wise Performance Analysis
- Customer Analysis
- Monthly Sales Trend Analysis
- Discount vs Profit Analysis
- Correlation Analysis
- Business Insights Generation

---

## Key Insights
- Technology category generated the highest sales.
- Higher discounts often resulted in lower profits.
- Certain states contributed significantly to overall revenue.
- A small group of customers generated a large share of sales.
- Monthly trends revealed peak business periods.
